## 💻 開発実績（プロダクト）

### 1. バンド情報特化型 RAG（検索拡張生成）システム
バンドの公式ホームページからライブ情報のみをスクレイピング等で自動抽出し、そのデータをもとに的確な回答を生成するRAGシステムを構築しました。

* **開発の背景:** 散らばっているライブ情報を1つにまとめ、ユーザーが自然言語で簡単にスケジュールを問い合わせられるようにするため。
* **使用技術:** Python, LangChain, Chroma, OpenAIAPI
* **担当・工夫した点:** 必要な「ライブ情報」だけを精度高く抽出してデータベースに構築する仕組み。
* **URL:** https://github.com/awa4000/RAG_hey
